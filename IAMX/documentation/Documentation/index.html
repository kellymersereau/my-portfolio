A) HTML Structure - top

The template is nicely responsive layout made with Bootstrap framework. All the content is within section. All the part of the template is divided with section. Main infrastructure is same into the all section. All contents are rounded with div, id and class. Most of the class and id are used according to Bootstrap. Here is the general structure.
